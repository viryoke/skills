## 4.2 4+1视图（按需选用）

> [!NOTE]
> 4+1视图按需选用，删除未使用的视图占位符。以下保留最常用的 3 种视图模板。
> 如需场景视图（用例图）、逻辑视图（类图）、物理视图（部署图）、状态变迁（状态图），
> 参考 `references/diagram_guide.md` 中对应示例添加。
> 统一使用 PlantUML 格式，不使用 Mermaid。

### 过程视图（时序图）— 多模块交互时必选

```plantuml
@startuml ProcessView
autonumber

actor "用户" as U
participant "模块A" as A
participant "模块B" as B

U -> A: 请求
A -> B: 调用
B --> A: 返回
A --> U: 响应
@enduml
```

### 开发视图（组件图）— 模块拆分时必选

```plantuml
@startuml DevelopmentView
skinparam componentStyle rectangle

package "子系统" {
    [组件A] as CA
    [组件B] as CB
}
CA --> CB : 依赖
@enduml
```

### 流程图（PlantUML Activity）— 业务流程时选用

```plantuml
@startuml FlowChart
start
:开始;
if (判断条件?) then (满足)
    :处理流程1;
else (不满足)
    :处理流程2;
endif
:结束;
stop
@enduml
```
